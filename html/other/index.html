This page-set contains the documentation for three related distributions:

Geo::Point
points, lines, surfaces, spaces which not only have a coordinate, but also knowledge about geographical transformations.
Geo::Proj4
XS wrapper around the Open Source libproj, which is able to do complex geographical transformations.
Math::Polygon
used to do simple mathematical transformations on shapes.
all manuals
List of all related manuals.
all methods
overview of all methods, arranged alphabetically
all diagnostics
list of all diagnostics which can be produced by this module. Often useful to determine which method is complaining about a mistake.
all details
Some manual pages contain a DETAILS chapter which explains all kinds of facts about the general background, implementation specifics, and examples of complex use of the items. Looking through the full list of details may help to find a solution.

Warning

The module is still quite young and therefore due for rapid change. The common structure will stand, but the capabilities will grow fast. I will do my best to be backwards compatible.
