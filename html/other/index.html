Geo-information related

This page-set contains the documentation for geography related distributions:

  • Geo::Point,
  • Geo::Proj4,
  • Geo::WKT,
  • Geo::GML,
  • Geo::KML (discontinued),
  • Geo::ISO19139,
  • Geo::EOP,
  • Geo::Format::Envisat, and
  • Math::Polygon.

Some of those use the XML format, and so for convenience some XML::Compile modules are added in the documentation tree.

Pick an entry point into the manuals:

all manuals
List of all related manuals.
all methods
overview of all methods, arranged alphabetically
all diagnostics
list of all diagnostics which can be produced by these modules. Often useful to determine which method is complaining about a mistake.
all details
Some manual pages contain a DETAILS chapter which explains all kinds of facts about the general background, implementation specifics, and examples of complex use of the items. Looking through the full list of details may help to find a solution.
